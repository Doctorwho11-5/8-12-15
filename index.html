<pre> Tag



<pre> the "pre" tag. This tag is used for typing things almost EXACTLY how you want them. Whatever you type 
on the Notepad, it will type on the webpage. I am using pre right now. Every space, new line, and punctuation 
mark will record how you type it.Although it will change a little bit to fit the page size.

Hosting



Now as for hosting, you can pay HostGator for 1 penny a month until the price goes up and OH NO! it's $50 now. 
Now you don't have money. OR, you can host it for FREE like me on github.com.
Now this takes some time to figure out and you may get seriously frustrated, but eventually, it turns 
out BEAUTIFUL. Now you still want to use the same thing I told you before, where you do it on 
notepad. But then you want to copy and paste the notepad codes INTO your website code section.


First sign up for Github.com, then click the cat icon in the upper left when you're done reading the tutorial. 
Then, click "new repository" and name it with whatever you like, except it has to be http://username.github.io
Where username is whatever you want it to be, you just can't use certain punctuations. 
Now, create the repository and make a README file. A README is just a document to inform people about the webpage.
once you're done, make a new file called index.html by clicking the '+' button next to your repository name. copy and paste
your file into the index.html file and click "commit new file". Then, you can go to http://username.github.io
and you're webpage is done! 
You can always edit your webpage and fine new codes to make your website amazing.
Now go make yourself a website.
